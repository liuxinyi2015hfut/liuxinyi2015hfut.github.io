jQuery

发表于 2016-11-21 | 分类于 JavaScript ， jQuery

原生JS封装成的一个库，zepto是jQuery的移动端小型化版本

版本

  • jquery-1.xxx （第一个版本，兼容所有浏览器）
  • jquery-2.xxx （第二个版本，在IE6，7，8下不兼容）
  • jquery-3.xxx （第三个版本，在IE6，7，8下不兼容）

jQuery原理

selector : 是个参数，选择器
context: 上下文
window.jQuery = window.$ = jQuery;
jQuery 是一个方法，又是一个类
$(".first") 是一个jQuery实例

extend扩展jQuery 方法

jQuery 的extend 方法 ，是用来扩展jQuery 的方法

jQuery选择器

DOM 是元素节点组成的

原声js中获取元素对象的方式
document.getElementsByClassName("first") 在IE6、7、8下不兼容
document.getElementById("div");兼容IE6、7、8
document.getElementsByTagName("*") （获取document下所有元素对象）兼容IE6、7、8

基本选择器

参数 描述
"#first" 通过ID名来获取
".first" 通过类名来获取
"div" 通过标签名来获取
"#first,.oUl" 并集选择

层级选择器

参数 描述
#first .oUl 获取后代所有元素对象
"#first>.oUl" 获取下一级的元素对象（紧邻的父子关系）
"#first+" 获取同级下紧邻的弟弟元素对象
".first+div" 获取同级下紧邻的div标签的弟弟元素对象
".first~ " 获取同级下所有的弟弟元素对象
".first~div" 获取同级下所有div标签的弟弟元素对象

基本筛选器

排除

参数 描述
":not(selector)" 去除所有与给定选择器匹配的元素

操作索引

参数 描述
".first:first" 第一个class名为first的元素
".first:last" 最后一个class名为first的元素
".first:even" 索引为偶数的class名为first的元素
".first:odd" 索引为奇数的class名为first的元素
".first:eq(0)" 索引为0的class名为first的元素
".first:gt(2)" 索引大于2的class名为first的元素
".first:lt(2)" 索引小于2的class名为first的元素

内容

参数 描述
":contains('珠峰')" 获取所有元素内容包含“珠峰”的元素
":empty" 获取所有不包含子元素或文本的空元素
":parent" 获取所有包含子元素或文本的非空元素
":has(.first)" 获取所有后代元素中包含类名为first元素的元素

可见性

参数 描述
":hidden" 获取所有不可见元素，type为hidden
":visible" 获取所有可见元素

属性

参数 描述
"[attribute]" 获取带有attribute属性的元素
"[attribute=value]" 获取attribute属性值为value的元素
"[attribute^=value]" 获取attribute属性值以value开始的元素
"[attribute$=value]" 获取attribute属性值以value结尾的元素
"[attribute*=value]“ 获取attribute属性值包含value的元素

子元素

参数 描述
":first-child" 查找父元素中第一个子元素是某类型的的元素
":first-of-type" 查找父元素中第一个某类型的子元素
":last-child" 查找父元素中最后一个子元素是某类型的的元素
":last-of-type" 查找父元素中最后一个某类型的子元素
":nth-child" 查找父元素中第n个子元素是某类型的的元素
":nth-of-type" 查找父元素中第n个某类型的子元素
":nth-last-child" 查找父元素中倒数n个子元素是某类型的的元素
":nth-of-type" 查找父元素中倒数n个某类型的子元素
":only-child" 查找父元素中唯一的子元素是某类型的的元素
":only-of-type" 查找父元素中唯一的某类型的子元素

其他

参数 描述
":header" 获取所有h1~h6的标题元素
":animated" 匹配所有正在执行动画效果的元素
":focus" 匹配当前获取焦点的元素
":image" 匹配所有图像域

zepto(移动端的小型的jQuery)

zepto和jQuery的不同

jQuery zepto
在PC端使用，里面包含了大量的对于IE低版本浏览器的兼容处理 应用于移动端开发，对IE版本浏览器不支持
提供了很多的选择器，以及DOM的操作方法 实现了部分常用的选择器和方法
动画的方法：animate、hide、show、toggle、slide、Down、slideUp、slideToggle、fadeIn、fadeOut、fadeToggle… 只有animate

zepto常用的事件操作

tap、singleTap、doubleTap、longTap、swipe、swipeUp、swipeDown、swipeLeft、swipeRight
