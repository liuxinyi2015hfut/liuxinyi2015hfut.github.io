制作esj渲染的模板，在原有的HTML结果基础上把我们的数据嵌套进来

1、制作esj渲染模板

  • js代码(流程控制标签)：<%js代码%>
  • 需要输出的内容：
    • <%=变量%>（原文输出HTML标签）
    • <%- %>（HTML会被浏览器解析）
  • 注释标签: <%# %>
  • 对标记进行转义: %
  • 去掉没用的空格: -%>

ejs中的逻辑代码全部用JavaScript

2、嵌套数据

  • 1、获取ejs模板——curEle.innerHTML
  • 2、使用ejs的render方法将数据和模板结构结合拼得到拼接后的字符串——ejs.render(模板字符串,数据)
  • 3、将拼接后的字符串放入页面元素容器中——容器元素.innerHTml=拼接后的字符串
